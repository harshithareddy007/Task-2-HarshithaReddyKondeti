# Project 2 - The Blueprint (Wireframing) 🏗

## DecodeLabs UI/UX Internship

### Overview

This project focused on information hierarchy and user flow.

The goal was to create a clear homepage structure before adding visual design.

---

## Scenario

An e-commerce company was losing customers during checkout because users found the interface cluttered and confusing.

The company needed a cleaner homepage structure with improved navigation and clearer calls-to-action.

---

## My Approach

I designed a low-fidelity homepage wireframe using grayscale elements only.

The wireframe includes:

* Header Navigation
* Hero Section
* Categories Section
* Featured Products
* Trust & Reviews Section
* Footer

I also labeled the sections to clearly communicate the structure and hierarchy.

---

## Deliverables

* Homepage Wireframe
* Information Hierarchy
* CTA Placement

---

## Tools Used

* Figma

---

## Key Learning

Wireframes helped me understand how structure and hierarchy influence user experience before visual styling begins.
